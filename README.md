# momukgi
# 🍽️ 오늘 인후에서 뭐먹지?

## 실행 방법

1. 프로젝트 폴더(`momukgi`)를 다운로드합니다.
2. 폴더 안의 `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 열거나, VS Code에서 Live Server 확장을 설치한 뒤 **Go Live** 버튼을 클릭합니다.


## 작품 소개

매일 반복되는 "오늘 뭐 먹지?"라는 고민을 해결해주는 랜덤 인후 메뉴 추천 웹앱입니다.  
버튼 하나만 누르면 한식, 일식, 중식, 양식, 분식, 패스트푸드 중에서 랜덤으로 메뉴를 추천해줍니다.  
HTML, CSS, JavaScript만으로 만든 순수 웹 페이지입니다.


## 주요 기능

- **랜덤 메뉴 추천**: 버튼을 누를 때마다 메뉴 목록에서 랜덤으로 하나를 뽑아 화면에 표시합니다.
- **카테고리 표시**: 추천된 메뉴와 함께 한식/일식/중식/양식/분식/패스트푸드 카테고리를 함께 보여줍니다.
- **다양한 메뉴 목록**: 인후에 있는 맛집들을 모아서 저장해놨습니다.


## 본인이 신경 쓴 기술적 포인트

- `Math.random()`과 `Math.floor()`를 조합해 메뉴 배열에서 랜덤 인덱스를 뽑는 로직을 직접 구현했습니다.
- 메뉴 데이터를 단순 문자열이 아닌 `{ name, cat }` 객체 형태로 구조화해서 이름과 카테고리를 함께 관리했습니다.
- `document.getElementById()`로 HTML 요소를 JS에서 직접 찾아 텍스트를 동적으로 변경했습니다.
- CSS의 `button:hover` 속성을 활용해 마우스를 올렸을 때 버튼 색이 바뀌는 인터랙션을 추가했습니다.

## 기타

처음으로 HTML, CSS, JavaScript를 배우며 만든 첫 번째 웹 프로젝트입니다.  
코드 한 줄씩 직접 입력하고 오타도 디버깅하면서 웹이 동작하는 원리를 익혔습니다.
